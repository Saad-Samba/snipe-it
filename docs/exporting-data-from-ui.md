# Exporting Data from the UI

This guideline outlines the different ways to export data directly from the Snipe-IT user interface, along with tips for choosing the right export method.

---

## 1) Export from list views (table export)
Most list pages (Assets, Accessories, Users, etc.) use the table toolbar to export the data currently shown on-screen. Use the **Export** button in the table toolbar to download data in the format you need. Available formats include Excel (XLSX), CSV, PDF, JSON, XML, TXT, SQL, and DOC.

**When to use it**
- You want a quick export of the list you’re already viewing.
- You need to export the result of specific filters, searches, or column selections.

**Tips**
- Apply filters and search terms before exporting to keep your export focused.
- Use the column picker to hide columns you don’t need before exporting.

## 2) Export after tailoring the view (filters + columns)
If you need a more targeted export, build the view first:
1. Apply filters (status, category, location, etc.).
2. Adjust visible columns.
3. Export from the toolbar.

This ensures the export reflects the exact subset and column layout you need.

## 3) Print-to-PDF and other table toolbar actions
Many list views also include **Print** and other toolbar actions next to **Export**. If you need a printable snapshot or PDF for quick sharing, use **Print** in the toolbar (your browser can save the result as a PDF).

**When to use it**
- You need a human-readable report for meetings or audits.
- You want a PDF snapshot that matches the on-screen view.

## 4) Export from dedicated report screens
Some reports have their own export or download options. Use report-specific export tools when you need the data as presented by that report.

**When to use it**
- You need a report-specific layout or summary.
- The report includes calculated fields not shown in standard lists.

## Choosing the right export method
- **List view export** for fast, ad-hoc exports.
- **Filtered view export** for targeted datasets.
- **Print-to-PDF** for human-friendly snapshots.
- **Report exports** for specialized or summarized data.
